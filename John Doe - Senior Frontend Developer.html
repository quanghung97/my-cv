Hướng dẫn thêm Tech Icon:
1. Click Browse Icons để tìm icon
2. Copy tên class icon (vd: devicon-react-original)
3. Thêm vào JSON với format:
"key": { "icon": "devicon-xxx-plain", "iconColor": "#color", "bgColor": "#bg", "textColor": "#text", "label": "Name" }
Projects - Tech Badge:
• "tech": Mảng các key từ tab Tech Icons (vd: react, typescript, nodejs)
• Xem danh sách key có sẵn tại tab Tech Icons
• Thêm tech mới: Vào tab Tech Icons → Thêm key mới
Skills - Tech Badge:
• Mỗi category chứa mảng các key từ tab Tech Icons
• Xem danh sách key có sẵn tại tab Tech Icons
• Thêm tech mới: Vào tab Tech Icons → Thêm key mới
Quản lý Sections:
• Thay đổi thứ tự: Đổi giá trị "order" (số nhỏ = lên trước)
• Ẩn/hiện section: Đổi "enabled": true/false
• Thêm section mới: Thêm vào customSections:
{ "id": "mySection", "title": "My Section", "icon": "skills", "enabled": true, "order": 1.5, "items": ["Item 1"] }
• Icon: Dùng key từ tab Section Icons (vd: overview, skills, education)
Section Icons (SVG Path):
1. Vào Heroicons hoặc Feather Icons
2. Chọn icon → Copy SVG → Lấy phần <path d="...">
3. Thêm vào JSON: "myIcon": "<path ... />"
4. Dùng key "myIcon" trong Sections config
